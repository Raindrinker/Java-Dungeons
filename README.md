# Java-GameStuff
Various bits of code for games in Java

###LIST OF STUFFS

* **DungeonGenerator**

Creates a dungeon in the format of a 2D matrix where 0s are walls and 1s are walkable tiles.
Has various configuration values (size, number of rooms, room size, etc.).
